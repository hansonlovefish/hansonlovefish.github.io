Appium简介

Appium是一个开源的自动化测试框架，是为了原生使用、混合和移动Web应用程序测试。他是基于iOS、Android、和Windows应用程序

定义

首先我们需要理解知道自动化测试的路线，分为web自动化测试和移动自动化测试。其实，我们还可以理解为Appium其实WebDrive的一个扩展而成的。

  • web auto-testing路线：编程语言基础—>测试框架—>webdriver API—>开发自动化测试项目
  • mobile auto-testing路线：程语言基础—>测试框架—>android/IOS开发测试基础—->appium API —–>开发移动自动化项目

Appium优势

由于移动端自动化测试也是当前热门测试技术，所以也具有一定的含金量。

  • 支持多平台的测试
  • 对语编程语言没有局限：java、pyhton、php等等
  • Appium用于Client/Server的设计模式。它接受来自客户端的连接，监听命令并在移动设备上执行，答复 HTTP 响应来描述执行结果,只要client能够发送http请求给server，那么的话client用什么语言来实现都是可以的，这就是如何做到支持多语言的原因

Appium为了实现以上的理念，设计出了一套特殊的框架（把系统本身的框架包装进一套API），这样Appium不需要把Appium特定的或者第三方的代码，就可以达到测试使用的应用和最终发布的应用并无不一样。所以学过selenium的更容易上手。
